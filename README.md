The rapid spread of misinformation on social media has created the need for fast and reliable fake news detection. This project proposes a fact-based fake news detection system that verifies news content by comparing  it  with  trusted  and  authoritative  sources.  The  system accepts text or image inputs, using OCR to extract text from images. Verified news data is continuously collected from reliable sources and stored in a local database. Based on this verification, news is classified as Real or Potentially Fake, ensuring higher reliability and transparency compared to traditional machine-learning approaches.
